# Privacy Policy

Last updated: 2026-03-10

This repository is for small experimental and educational projects.

By default, it does not intentionally collect, sell, or share personal data from end users.

Some projects may process local inputs, logs, cached files, or generated outputs provided by the person running the code. If a project uses third-party services, those services are governed by their own privacy policies.

This repository is provided on a best-effort basis and is not intended as a production data-processing service.
